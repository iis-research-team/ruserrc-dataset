# Cross-domain RUSERRC

The dataset contains texts and annotation for:
* Terms extraction;
* Relations extraction;
* Aspects extraction


## Terms

Mark up with scientific terms is provided in BIO format in the `terms/` directory.


## Relations

**Relation.csv** is the file with information about relations for the terms.

Mark up contains entities (terms) and semantic relations between them.

Info about dataset:
- Number of texts: 182
- Number of tokens: 15 789
- Number of terms: 5 181
- Number of relations: 222

Relations by type:
- USAGE: 89
- ISA: 81
- PART_OF: 52


## Aspects

Mark up with aspects is provided in the `aspects\` directory.
